# Online Rogue Beta Deploy Fixed

这是修复版，重点解决：
- 朋友加入房间没有明显反馈
- 房间码大小写/空格导致加入失败
- 前端看起来“没跳页”但实际是 join 没成功
- 缺少连接状态提示

## 本地运行

```bash
npm install
npm start
```

## Railway / Render

直接替换原项目代码后重新部署即可。


## 本次更新
- 房间等待时可自由移动
- 单人也能准备后开始游戏
- 怪物改为原创像素贴图风，不使用第三方游戏官方素材
- 升级出现时角色仍可继续移动


## 多状态动画版
- 角色状态：待机 / 行走 / 攻击 / 受击 / 死亡
- 怪物状态：待机 / 行走 / 攻击 / 受击 / 死亡
- 使用 players_states.png 与 enemies_states.png 进行状态切换
- 受击状态通过客户端检测血量变化触发


## 方向动画版
- 在多状态动画基础上加入左右朝向
- 角色与怪物会根据 aimX / 移动方向自动翻转
- 攻击与移动朝向绑定，观感更接近正式游戏


## 装备系统（第一版）
- 精英/Boss 与部分怪物会掉落装备
- 三个装备槽：武器 / 护甲 / 遗物
- 靠近自动拾取并替换同槽位装备
- 装备提供攻击、攻速、移速、生命等加成


## 装备品质系统增强
- 新增装备图标 equipment_icons.png
- 品质颜色：普通 / 稀有 / 史诗
- 精英/Boss 掉装更有辨识度
- HUD 新增装备面板与拾取提示


## 背包 + 商店 + 词条版
- 新增 6 格背包，可装备或出售背包中的装备
- 装备增加随机词条（如 锋利 / 迅捷 / 坚韧）
- 新增商人面板，可花金币购买随机装备
- 装备栏、背包、商店都在 HUD 中显示


## 版本 1：装备比较与拾取反馈
- 新增拾取对比面板
- 靠近掉落时自动比较当前同槽位装备
- 显示攻击 / 攻速 / 移速 / 生命的增减


## 版本 2：套装与铁匠
- 装备新增套装归属（狂战 / 守御 / 奥术 / 夜行）
- 两件同套装可激活额外加成
- 新增铁匠重铸：消耗金币洗当前装备属性，保留品质


## 版本 3：营地元成长
- 每局结束获得魂晶（本地持久化）
- 菜单新增营地成长面板
- 可永久升级初始攻击 / 生命 / 移速
- 创建或加入房间时会带入当前元成长配置


## V4 试玩打磨版
- 新增波次进度条与顶部金币显示
- 新增小地图，显示队友、敌人和装备掉落
- Ready 按钮会根据房间状态动态显示
- 本地暂停功能（P 键）
- 拾取提示改为短历史列表，避免一闪而过
